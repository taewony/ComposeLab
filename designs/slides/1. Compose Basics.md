## 선언적 UI 패러다임
### 명령형 vs 선언적 접근법 비교

```kotlin
// 명령형 접근 (XML + View)
textView.text = "Hello"
textView.visibility = View.VISIBLE

// 선언적 접근 (Compose)
@Composable
fun Greeting() {
    Text("Hello") // UI는 상태의 함수
}
